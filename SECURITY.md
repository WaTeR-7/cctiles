# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in CC-Tiles, please **do not** open a
public issue for it. Instead, report it privately using
[GitHub's private vulnerability reporting](https://github.com/WaTeR-7/cctiles/security/advisories/new)
for this repository.

We'll do our best to acknowledge reports promptly and keep you updated as the
issue is investigated and resolved.

## Supported Versions

This project has not yet reached a stable 1.0 release. Until then, only the
latest version on the `main` branch is supported.
